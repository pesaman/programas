String

Los strings o cadenas de texto son secuencias de caracteres. La sintaxis para definirlos es utilizando comillas simples o comillas dobles.


Array

Es una lista de datos ordenados numéricamente. Por ejemplo, una lista( array ) de los numeros del 5 al 7 sería: primero el 5,segundo el 6 y tercero el 7. En ruby la lista la escribiríamos : [5,6,7] y es una lista de tipo numérico. Una lista( array ) de amigos sería : ["pepe","juan","pedro"], es una lista de tipo cadena donde primer amigo sería "pepe", el segundo "juan" y el tercero "pedro".
